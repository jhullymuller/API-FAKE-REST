US001 :: Criação de uma nova atividade

Como usuario da aplicação 
Quero cadastrar novas atividades
Para consultar as atividades disponiveis em sistema

Criterios de Aceitação 
- A resposta deve retornar HTTP 200.
-Deve retornar uma lista com todas as atividades cadastradas com apenas 
    - ID
    - Titulo
    - Data e hora da criação
